# 🛒 Online Retail Sales Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **Online Retail Sales Dashboard** built using **Microsoft Power BI**. The dashboard provides valuable business insights by analyzing sales performance, customer purchasing behavior, regional sales distribution, and product category trends.

The objective of this project is to help businesses make data-driven decisions through effective data visualization and KPI tracking.

---

## 📊 Dashboard Features

- 📈 Total Revenue Analysis
- 🛍️ Product Category Performance
- 🌍 Region-wise Sales Analysis
- 👥 Customer Purchase Analysis
- 💳 Payment Method Distribution
- 🔄 Repeat vs New Customer Analysis
- 📅 Sales Trend by Date
- 📦 Quantity Sold Analysis
- 🎯 Interactive Filters (Slicers)

---

## 🗂 Dataset Information

The dataset contains retail transaction details with the following fields:

| Column | Description |
|---------|-------------|
| OrderID | Unique Order ID |
| CustomerID | Customer Identifier |
| OrderDate | Date of Purchase |
| Category | Product Category |
| Region | Sales Region |
| Quantity | Number of Products Sold |
| UnitPrice | Price per Product |
| Revenue | Total Sales Amount |
| PaymentMethod | Payment Type |
| Purchase Type | New Customer / Repeat Customer |
